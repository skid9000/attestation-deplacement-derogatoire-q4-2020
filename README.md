# Générateur de certificat de déplacement

## Développer

### Installer le projet

```console
git clone https://github.com/LAB-MI/attestation-deplacement-derogatoire-q4-2020.git
cd attestation-deplacement-derogatoire-q4-2020
npm i
npm start
```

### Pour éditer le PDF de référence

Go to : src/certificate.pdf

### Pour pré-remplir le formulaire

Go to : src/form-data.json

(Attention, il faudra redémarrer le projet en cas d'edit.)
